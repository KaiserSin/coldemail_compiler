# Cold Email Generator

Скрипт для генерации персонализированных cold email на основе списка компаний.  
Использует OpenAI API для формирования вступлений писем и сохраняет результат в готовую таблицу.

---

## Структура проекта

- **`companies_list.csv`**  
  Входной список компаний.  
  Содержит поля с данными компании: название, ИНН, дата регистрации, руководитель, телефоны, e-mail, сайт, основной вид деятельности и др.  
  Этот файл можно заменить на свой список компаний с аналогичными заголовками.

- **`reader.py`**  
  Обрабатывает исходный список:
  - оставляет только необходимые колонки,
  - собирает все e-mail в один столбец,
  - сортирует и подготавливает данные.  
  Результат сохраняется в `companies_filtered.csv`.

- **`letters.py`**  
  Основной исполняемый файл проекта.  
  Выполняет:
  1. Запуск подготовки данных (`reader.py`).
  2. Формирование персонализированных запросов для каждой компании.
  3. Асинхронные обращения к OpenAI API.
  4. Генерацию текста письма: персонализированное начало + заготовленный шаблон.
  5. Сохранение результата в `emails_ready.csv`.
